Service

基本概念

Service表示OSS提供给用户的虚拟存储空间，在这个空间中，用户最多可以拥有10个Bucket

Service的操作

获取Bucket列表

操作封装在GetServiceTask中。通过该Task可以得到一个Bucket列表。具体代码如下：

task = new GetServiceTask();
this.initKey(task);
List<Bucket> buckets = task.getResult();
