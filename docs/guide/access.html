错误响应

Access Key

作为OSS注册用户，系统会分配给用户一堆AccessKeyID和AccessKeySecret，称为ID对。

在使用SDK进行开发的过程中，需要用到ID对的地方主要有：

  • Task对象，每一个Task对象都需要ID对来对服务器进行访问
  • OSSClient对象，记录ID对来实例化任务然后对服务器进行访问

用户只需要将ID对交给Task或者OSSClient对象即可，中间复杂的签名、验证规则SDK都已经封装过了，不需要用户再关注细节。
