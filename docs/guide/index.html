API使用指南概述

欢迎使用阿里云开放存储服务Android SDK，这里首先对整个指南进行一个概述，方便各位开发者更好地使用SDK进行开发。

OSSClient

面向开发者友好的OSS服务编程接口，它能够胜任简单而基本的任务，推荐初次使用SDK或者应用场景并不复杂的开发者使用。

Task

任务编程接口，每一个具体任务对应一种OSS HTTP REST API，对于需求比较复杂、定制性较高的开发者推荐基于Task进行开发

基本组件

无论基于哪一种SDK的使用方式，都需要对SDK的基本组件元素有一定了解，这里对OSS中每一种概念都进行了介绍，并且描述了SDK中的对应元素，同时简单引入了一些示例代码

访问控制

SDK已经对权限、验证、签名等等机制做了良好的封装，开发者不再需要关系具体实现细节了

实用工具

对于一些复杂的API来说，可能Task对象处理起来还是比较复杂的，为了降低复杂性，SDK提供了分页工具抽象出分页逻辑，使得开发者能够很方便的处理分页。

分块上传是OSS为一些场景例如断点续传、网络状况很差等等而设置的，SDK也对此进行了封装

加密、压缩也是SDK提供的比较有用而方便的功能，用户通过压缩存储可以节约流量，通过加密存储可以提高安全性
